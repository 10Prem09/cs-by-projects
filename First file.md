# How to handle git 

- To add/remove the new changes we can use the `git add` or the `git remove` command. 
- To commit and add comment after adding files, whose changes we want reflected, we use `git commit -m "First Commit"` 
- To reflect in github, we do `git push origin main`, or any branch which we want to update. 
- To Download latest, `git pull origin main`. 
- To Check Status, `git status`
- To Check the history of commits, `git log`
